

# 📦 Predicción de Ingresos: Amazon Sales Dataset
Este proyecto final aplica técnicas de **Machine Learning** para predecir el ingreso total de ventas basándose en el comportamiento histórico de la plataforma.

## 📝 Resumen del Proceso
1. **EDA:** Análisis exploratorio completo (visualizaciones con Seaborn).
2. **Feature Selection:** Selección de variables mediante matriz de correlación para reducir dimensionalidad.
3. **Modelado:** Entrenamiento de un algoritmo de **Random Forest Regressor**.
4. **Validación:** Obtuvimos un **R² de 1.0000** y un **MAE de 0.54**, demostrando una precisión altísima.

## 🚀 Hipótesis
* **H1 (Alternativa):** Las variables como precio, cantidad y categoría permiten predecir con exactitud el ingreso total. (VALIDADA ✅)
* **H0 (Nula):** No existe relación entre las variables y el ingreso. (RECHAZADA ❌)
